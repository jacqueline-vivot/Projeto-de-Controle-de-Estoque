# Projeto-de-Controle-de-Estoque
Sistema de Controle de Estoque, entrada e saída de produtos


##  Descrição do Projeto
Este projeto faz parte da disciplina de Engenharia de Software e tem como objetivo aplicar metodologias ágeis no desenvolvimento de um sistema simples de gerenciamento de um estoque.  
O sistema foi desenvolvido para uma empresa fictícia chamada **Choco D. Late**, contratada por uma startup de logística que deseja acompanhar seu fluxo de trabalho em tempo real.

---

##  Objetivo
Criar um sistema que permita:
- Auxiliar no controle de estoque.
- Ter um controle de preços e prazo de validade.
- Possibilidade de consultar o histórico dos produtos.

---

##  Escopo do Sistema
Funcionalidades básicas:
- CRUD de tarefas (Criar, Listar, Atualizar, Excluir).
- Classificação das tarefas em três status:
  -  A Fazer
  -  Em Progresso
  -  Concluído

Tecnologias utilizadas (exemplo):
- Linguagem: Python (tkinter)
- Controle de versão: Git / GitHub
- Integração contínua: GitHub Actions
- Metodologia: Kanban

---

## ⚙️ Metodologia de Desenvolvimento
O projeto segue princípios ágeis, combinando e Kanban:

- **Kanban no GitHub Projects:** dividido em três colunas:
  - **A Fazer** → tarefas planejadas
  - **Em Progresso** → tarefas em andamento
  - **Concluído** → tarefas finalizadas
